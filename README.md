# 🌐 PortSwigger Web Security Academy - Writeups

This repository contains my hands-on practice and writeups from PortSwigger Web Security Academy.

I use this platform to learn real-world Web Application Security vulnerabilities and understand how attackers exploit them and how to fix them.

---

## 🎯 Objective

- Understand real-world web vulnerabilities in depth
- Practice manual penetration testing techniques
- Learn exploitation and impact of security flaws
- Improve VAPT (Vulnerability Assessment & Penetration Testing) skills
- Prepare for Junior Security Analyst / Pentester roles

---

## 🧠 Topics Covered

### 🛡️ OWASP Top 10 Vulnerabilities
- Cross-Site Scripting (XSS)
- SQL Injection
- Cross-Site Request Forgery (CSRF)
- Insecure Direct Object Reference (IDOR)
- Authentication & Session Flaws
- Access Control Vulnerabilities
- File Upload Vulnerabilities
- Command Injection Basics

---

## 🧪 Hands-on Practice

Each lab is solved in a controlled environment provided by PortSwigger Web Security Academy.

For every lab, I focus on:
- Understanding the vulnerability
- Identifying attack points
- Exploiting the issue step-by-step
- Understanding real-world impact
- Learning how to fix the issue
